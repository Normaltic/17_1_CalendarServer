# 17_CalendarProject
Calendar Project for Capstone design <br />
17년 상반기 캡스톤 디자인 관련 MongoDB 연동 express API 서버.<br />

## API
* 계정 등록
* 로그인 ( 토큰 발급 )
* 계정 확인
* 일정 등록(or 공유)
* 일정 수정
* 일정 삭제
* (공유)일정 조회
* 조율일정 등록
* 조율일정 수정
* 조율일정 삭제
* 조율일정 투표
* 조율일정 댓글 등록
* 조율일정=>공유일정
* 조율일정 리스트 조회
